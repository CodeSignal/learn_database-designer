Overview

This page explains how to use the DB Schema Designer: create tables, add fields, set keys, connect tables, and save your work.

Pan & Zoom

  • Use the mouse wheel to zoom in/out. The view zooms around your cursor.
  • Drag the empty background to pan the canvas.

Moving Tables

Grab the dotted handle in the table header to drag a table.

Adding & Editing Fields

  1. Select a table from the diagram.
  2. Use the Add Field form in the sidebar: name, type, nullable/default, primary/unique.
  3. Click a field in the diagram to edit it again.

Primary & Unique Keys

Mark a field as Primary key or Unique in the field form. Primary keys are shown with a key icon.

Foreign Keys

  1. Choose the From column, To table, and To column.
  2. Pick an On Delete action (e.g., CASCADE).
  3. Click Add FK. Arrows will appear between related tables.

To remove an FK for a field, select the field and use the Remove option (status text will confirm).

  • ESC - Close help modal
  • Mouse wheel - Zoom in/out
  • Drag background - Pan canvas
  • Click table - Select table
  • Click field - Edit field

Troubleshooting / FAQ

How do I delete a table?

Select the table and click the "Delete" button in the sidebar. This will also remove any foreign keys connected to the table.

Can I change the color of tables?

Yes! Select a table and use the "Table color" dropdown in the sidebar to choose from white, blue, green, or red.

How do I add images to the help?

Place image files in the help/img/ directory and reference them with relative paths like <img src="./img/example.png" alt="Description">

Is my work automatically saved?

Yes, the app automatically saves your schema every second to both localStorage and the server. You'll see "Auto-saved" status messages.
